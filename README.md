# Machine Learning Engineer Roadmap

This repository documents my six-month journey toward becoming a machine learning engineer.

## Main Goals

- Improve my Python programming independence
- Learn NumPy, pandas, visualization, and statistics
- Understand core machine learning algorithms
- Build two strong portfolio projects
- Learn model deployment using Streamlit and FastAPI
- Practise testing, Docker, and basic MLOps
- Prepare for machine learning and data-related internships

## Learning Method

Each topic follows this process:

1. Concept explanation
2. Worked example
3. Guided exercise
4. Independent exercise
5. Project application
6. Weekly review and quiz

## Repository Structure

- `01-topic-notes/` — concise Markdown notes
- `02-coding-exercises/` — Python and ML exercises
- `03-mini-projects/` — short learning projects
- `04-portfolio-projects/` — major portfolio projects
- `05-weekly-quizzes/` — weekly assessments
- `06-weak-topic-reviews/` — topics requiring more practice
- `07-career-preparation/` — résumé, interview, and job-preparation work
- `debugging-log.md` — errors, causes, fixes, and lessons
- `ai-assistance-log.md` — record of AI explanations, hints, debugging, and reviews
- `progress-tracker.md` — weekly progress and checkpoints
